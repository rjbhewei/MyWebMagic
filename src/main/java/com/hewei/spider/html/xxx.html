重庆

2009年11月，重庆市各大网站竞相爆出一年轻女孩的艳照，数量达数十张。还有该女孩身着工作服拍的生活照，工作服上的工号显示，她是重庆市某大型医院护士。由于无法核实图片上女孩的身份，她的真实身份还是个谜。
这些照片拍摄于多处室内，看上去有的在家中，有的在宾馆，均为同一年轻女性。其中大多为该女子的正面、侧面及背面的全裸、半裸照片。照片中，女子身穿护士服，制服上有某医院字样，护士服上还有其工号。
院方为减少此事对医院的负面影响，已向公安机关报案，并对上传有护士“艳照”的网页进行了处理。

胡雯靖

2009年8月，胡雯靖艳照门照片曝光，成为热议。2009年8月，一篇题为《和成都医科大护士系02级小美女胡雯靖性艳照曝光附闺蜜自拍照》的帖子在网上流行，成为热帖。帖子内容为一组性爱艳照，发帖人称女主角是成都医科大学护士系02级学生胡雯靖。

龙口

龙口门2009年9月23日，一段不道德的视频被放上了网上，被人们疯狂下载，而且标题更是涉及到疑似山东龙口中心医院女护士与当地税务局的某个干部，因此而在网络上火爆起来。更有网友跑去中心医院的百度贴吧发帖。经烟台都市网站长冰客王子介绍，视频中的男女主角确实说的是烟台口音，龙口是烟台的县级市，也说烟台话。烟台话，就是福山话，是烟台福山区的经典口音。
在龙口护士门事件中，男主角的年龄看起来只有三十三四岁左右，不可能担任局长要职。奥库传媒营销公司认为这是陈冠希综合征的结果，每一个大事件后面都会有人模仿，他们炒作的不是视频，是寂寞。

大连

2010年9月，大连某医院的某护士被男友把两人的亲密照发到网上，引发了“大连护士艳照门”的轩然大波，照片数量众多，甚至全裸无码令众网友跌掉眼镜！大连护士自拍的图片，尺度大胆，且有些少儿不宜的照片，所以受到了广大网友们的批判，纷纷带着仇恨的目光审视了这些照片，并给与了不雅照的定位。因此网友们称之为“大连护士门”或“大连护士艳照门”。[1] 

2事件影响编辑

女人的艳照属于隐私范畴，是不能随意在网上疯传的，一旦传出去，会对当事人造成不良影响。
网上传播他人隐私照属刑事犯罪。律师王洪认为，如果是女孩自己上传的照片，因为照片中有部分为黄色淫秽照片，情节严重的，可被处以刑事处罚;这种意识的缺失，则足以衍生网络道德的缺失。在网络错误文化引导与“无人知晓我是谁”的心理保护下，一方面，腐朽文化在盲目传播，另一方面，在盲目接受。于是，有些现象就变得肆无忌惮，无论艳照门，还是其他网络新闻事件，都出于网络道德的失范。

3社会舆论编辑

从陈冠希的艳照门开始，XX门成为社会讨论的热点问题，从此便开辟出了一条以XX门为主题的混乱的发展之路。这是一种盲从的心理。
重庆护士门，与诸如此类的还有寂寞门、摸奶门、混账门、做爱门等等，诸如此类的“门”大多与色情有关，一旦出现，在网上的点击率疯长。而与此同时，在这些门的背后竟是恶性插件和木马病毒，网友观看这些激情视频的同时，还将收到病毒的侵蚀。
